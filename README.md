# Мобильное приложение для определения погоды через сайт Gismeteo
- Пользователь может выбрать один из 6 городов (Москва, Санкт-Петербург, Нижний Новгород, Новосибирск, Тольятти, Сочи)
- Пользователь может выбрать период, за который ему интересно узнать погоду (Сегодня, Завтра, 3 дня, 10 дней)
# Запуск
- Склонировать репозиторий на свой локальный компьютер и открыть проект через AndroidStudio
- Собрать и запустить проект на эмуляторе, либо подключённом Android девайсе
